## VICEGRAM

`vicegram_bot.py` — отдельный бот-ловец для групповых чатов (по образцу Catcher):
ловит удалённые и изменённые сообщения, стикеры, файлы, гифки и другие медиа,
следит за исчезающими медиа, восстанавливает историю чата, считает "огоньки"
активности, банит скам-сообщения и поддерживает кастомные команды в чатах.

### Запуск

```bash
export VICEGRAM_BOT_TOKEN="токен_от_BotFather"
# необязательно: чат/канал для тихой проверки существования сообщений
# (по умолчанию используется личка с VICEGRAM_ADMIN_ID)
export VICEGRAM_LOG_CHAT_ID="-100..."
python3 vicegram_bot.py
```

Добавь бота в группу с правами администратора (минимум — удаление сообщений
и чтение истории чата) и напиши в группе `/settings`.

**Техническое ограничение Bot API:** Telegram никогда не присылает боту событие
"сообщение удалено". Ловля удалений реализована через периодическую тихую
проверку существования сообщения (задержка до ~30 секунд), а не мгновенно —
мгновенную ловлю даёт только MTProto-юзербот (Telethon/Pyrogram со входом по
номеру телефона), что требует отдельной архитектуры и `api_id`/`api_hash` с
my.telegram.org. Изменённые сообщения, медиа, анти-скам, огоньки и команды
работают штатно и мгновенно.
