## VICEGRAM

`vicegram_bot.py` — отдельный бот-ловец для групповых **и личных** чатов
(по образцу Catcher): ловит удалённые и изменённые сообщения, стикеры,
файлы, гифки и другие медиа, следит за исчезающими медиа, восстанавливает
историю чата, считает "огоньки" активности, банит скам-сообщения и
поддерживает кастомные команды в чатах.

### Запуск

```bash
export VICEGRAM_BOT_TOKEN="токен_от_BotFather"
# необязательно: чат/канал для тихой проверки существования сообщений в группах
# (по умолчанию используется личка с VICEGRAM_ADMIN_ID)
export VICEGRAM_LOG_CHAT_ID="-100..."
python3 vicegram_bot.py
```

Добавь бота в группу с правами администратора (минимум — удаление сообщений
и чтение истории чата) и напиши в группе `/settings`.

**Техническое ограничение Bot API в группах:** Telegram никогда не присылает
боту событие "сообщение удалено". Ловля удалений реализована через
периодическую тихую проверку существования сообщения (задержка до
~30 секунд), а не мгновенно. Изменённые сообщения, медиа, анти-скам,
огоньки и команды работают штатно и мгновенно.

### Личные чаты (ловля удалений в переписках с другими людьми)

Реализовано через официальную функцию Telegram **«Автоматизация чатов»**
(Telegram Business Bots) — никакого MTProto, номеров телефона или сторонних
библиотек не требуется. Пользователь подключает бота сам, прямо в своих
настройках Telegram:

1. Настройки → «Изменить»
2. «Автоматизация чатов»
3. Вписать username бота (например `@Helperchat_bot`) → «Добавить»
4. Выбрать, каким чатам бот будет доступен («Все личные чаты, кроме…» или
   «Только выбранные»)

После подключения Telegram сам присылает боту события `business_message`,
`edited_business_message` и `deleted_business_messages` — включая
настоящее мгновенное событие удаления **с указанием чата**, в отличие от
групп, поэтому здесь не нужны ни polling, ни эвристики. Никакого
дополнительного кода на сервере настраивать не нужно — это всё уже
работает из коробки в `vicegram_bot.py`.

Похоже, что эта функция может требовать Telegram Premium на аккаунте,
который подключает бота (это часть набора функций Telegram Business) —
если пункта «Автоматизация чатов» нет в настройках, скорее всего дело в
этом.
